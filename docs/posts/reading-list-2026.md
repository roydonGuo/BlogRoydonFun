# 2026 上半年阅读书单

转眼 2026 年已经过半，记录一下这半年读过的书。

## 技术类

### 《深入理解 Java 虚拟机》— 周志明

这本书是 Java 开发者的必读经典。第三版新增了模块化系统和 GraalVM 等内容，对 JVM 内存模型、垃圾回收算法、类加载机制讲解得非常透彻。

**推荐指数**：⭐⭐⭐⭐⭐

### 《CSS 揭秘》— Lea Verou

一本让人眼界大开的 CSS 技巧书，47 个实战案例涵盖了从选择器到布局的方方面面。读完对 CSS 的理解会上一个台阶。

**推荐指数**：⭐⭐⭐⭐

## 非技术类

### 《人类简史》— 尤瓦尔·赫拉利

从认知革命到科学革命，作者用宏大的视角梳理了人类历史的发展脉络。印象最深的是关于"虚构故事"的论述——国家、法律、货币，本质上都是人类共同相信的虚构故事。

**推荐指数**：⭐⭐⭐⭐⭐

### 《蛤蟆先生去看心理医生》— 罗伯特·戴博德

用童话故事的形式讲解心理学知识，轻松治愈。书中关于"儿童自我状态""父母自我状态""成人自我状态"的分析让人受益匪浅。

**推荐指数**：⭐⭐⭐⭐

## 阅读习惯分享

这半年我养成了一些不错的阅读习惯：

- **每天 30 分钟** — 通勤时间利用起来
- **读书笔记** — 读到精彩处随时记录，读完后整理成文
- **主题阅读** — 一段时间围绕一个主题读 2-3 本书，理解更深入

下半年继续加油 📚
